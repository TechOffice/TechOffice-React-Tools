# Nx

## React App Workspace

```
npx create-nx-workspace --preset=react
```

```
nx serve nx-react-app
```


```
npm run serve
```


## React Library
```
nx g @nrwl/react:library nx-react-component --buildable=true
```

```
nx build nx-react-component
```
